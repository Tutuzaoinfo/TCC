Smooth Invest — Dashboard e Simuladores Financeiros

O Smooth Invest é um conjunto de aplicativos em Streamlit voltados para visualização, análise e simulação de dados financeiros.
O projeto integra informações de diferentes fontes — como o Banco Central do Brasil e a API pública Brapi — e oferece calculadoras de juros, simuladores de investimento e dashboards dinâmicos.

O objetivo é fornecer um ambiente visual e interativo para o usuário acompanhar indicadores econômicos, simular aplicações financeiras e comparar bancos e investimentos de forma simples.

O projeto é composto por quatro aplicações principais:

App.py	Painel principal de bancos, investimentos e cotações de ações (Brapi API).
Dados governo.py	Dashboard de indicadores econômicos oficiais do Banco Central (SELIC, IPCA, CDI, Dólar).
Parcelas e juros.py	Calculadora de juros compostos e de parcelas.
Simulador de investimento.py	Simulador completo de investimentos em renda fixa com aportes e tributação.

O Smooth Invest é uma ferramenta educacional e demonstrativa, ideal para estudantes e investidores que desejam compreender conceitos de finanças pessoais, juros compostos e acompanhamento de indicadores econômicos.

Os dados utilizados provêm de fontes públicas e abertas, e as simulações têm caráter ilustrativo.
